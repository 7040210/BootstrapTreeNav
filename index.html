BootstrapTreeNav

http://morrissinger.github.io/BootstrapTreeNav

BootstrapTreeNav is a javascript plugin for Twitter Bootstrap 3 to help you create tree navigation menus, created and maintained by Morris Singer.

Quick start

Clone the repo, git clone git://github.com/morrissinger/BootstrapTreeNav.git, or download the latest release.
